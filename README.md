# AIS Frontend Tutor Components

## Project Overview
Shared UI components and React hooks for the PersonalAITutor ecosystem. This library ensures a consistent "companion" experience across both the Student and Teacher portals.

## Key Components
- **FloatingActionAgent**: The "R2D2" icon and chat bubble for students.
- **AssistantSidebar**: A persistent analytical panel for teachers.
- **TutorNotifications**: Shared toast system for proactive AI suggestions.

## Key Hooks
- **useTutorLink**: Shared session management and telemetry bridge.

## Getting Started
